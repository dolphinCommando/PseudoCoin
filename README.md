# PseudoCoin

## Purpose
PseudoCoin is a cryptocurrency trading “simulator.” Since the money is virtual, people new to crypto can learn how it works without losing their life savings.


## Technologies Used
React.js
MongoDB
Passport.js
Chart.js
Bootstrap
CSS
HTML
CoinBase’s API
Nodejs request
